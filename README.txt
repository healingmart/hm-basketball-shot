권장 저장소 이름
  hm-basketball-shot

GitHub에 올릴 파일
  index.html
  hm-basketball-shot.js

권장 배포 주소
  https://healingmart.github.io/hm-basketball-shot/

설정
1. GitHub에서 public 저장소 hm-basketball-shot 생성
2. index.html, hm-basketball-shot.js 두 파일을 저장소 최상단에 업로드
3. Settings > Pages
4. Build and deployment: Deploy from a branch
5. Branch: main / root
6. 저장 후 위 Pages 주소를 새 탭에서 먼저 실행
7. 게임이 단독 페이지에서 정상 실행되는 것을 확인
8. Blogger 기존 농구 게임 HTML/JS를 모두 삭제
9. blogger-iframe-code.txt 내용만 HTML 보기에서 삽입

중요
- Blogger에는 Three.js나 농구 게임 JS를 따로 넣지 않습니다.
- 기존 hm-basketball-shot.901fcee6.js 또는 96dbc01b.js 호출도 Blogger에서 제거합니다.
- iframe 안의 게임은 Blogger 페이지의 THREE/Kakao/광고/CSS와 별도 문서에서 실행됩니다.
- 사운드팩 로딩 실패는 게임 본체 실행을 막지 않도록 되어 있습니다.
